# ISO 27001 Risk Management Project

This project demonstrates a basic Governance, Risk, and Compliance (GRC) risk management process aligned with ISO 27001 concepts.

The project includes asset identification, risk assessment, control mapping, and security architecture documentation.

## Project Structure

Excel File:
GRC_IT_Asset_Register.xlsx

## Sheets Included

1. Asset_Register – List of company assets
2. Risk_Register – Identified security risks
3. Control_Mapping – Mapping risks to ISO 27001 controls
   

## Project Components

• IT Asset Register  
• Risk Register  
• Control Mapping  
• Risk Matrix  
• Security Architecture Diagram
