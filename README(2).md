# GRC Compliance Dashboard

This project demonstrates a GRC (Governance, Risk, and Compliance) dashboard built using Microsoft Excel.

## Features
- Compliance status tracking
- Risk level analysis
- Visual charts (Pie & Column)
- KPI: Compliance Percentage

## Sheets Included
- Control_List
- Compliance_Tracker
- Risk_Issue
- Dashboard

## Tools Used
- Microsoft Excel

## Purpose
To simulate real-world GRC monitoring and reporting.
